## 🛠️ **Technical Details**

### **Tools Used**
- **Microsoft Power BI Desktop**
- **DAX (Data Analysis Expressions)**
- **Power Query Editor**
- **Data Visualization**

### **Dataset**
- **Source:** Kaggle - Foodpanda Analysis Dataset 2025
- **Link:** https://www.kaggle.com/datasets/nabihazahid/foodpanda-analysis-dataset-2025
- **Features:** Order details, customer info, geographic data, payment metrics

### **Key DAX Measures**
```dax
Average Order Value = 
DIVIDE(
    SUM('Foodpanda Analysis Dataset'[total_revenue]), 
    COUNT('Foodpanda Analysis Dataset'[order_id]), 
    0
)
```
